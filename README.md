**QurbaniHat** is a sophisticated digital platform tailored for the livestock market, specifically focused on the Qurbani season. It combines modern web technologies to provide a seamless, secure, and interactive experience for both buyers and sellers of sacrificial animals.

---

## 🌟 Features

- **⚡ Next.js 15 Power:** Leverages the latest App Router, Server Components, and optimized rendering for peak performance.
- **🛡️ Secure Authentication:** Powered by **Better Auth**, ensuring robust session management and user data security.
- **💎 Premium UI Components:** Built with **HeroUI** (formerly NextUI) for a high-end, aesthetic, and glassmorphic look.
- **🎨 Interactive Experience:** Integrated with **Lottie Animations** to provide engaging feedback and modern loading states.
- **🔍 Smart Navigation:** Advanced routing and dynamic animal detail pages for better discoverability.
- **📱 Responsive Design:** Fully optimized for mobile, tablet, and desktop using **Tailwind CSS**.
- **🔔 Instant Feedback:** Real-time notifications via **React Toastify**.

---

## 🛠️ Tech Stack

- **Core Framework:** Next.js 15
- **Styling:** Tailwind CSS
- **UI Library:** HeroUI
- **Authentication:** Better Auth
- **Icons:** Lucide React
- **Animations:** Lottie React
- **Toast Notifications:** React Toastify
- **State Management:** React Hooks (useState, useEffect)
